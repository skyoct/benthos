Benthos logging prints to stdout (or stderr if your output is stdout) and is formatted as [logfmt](https://brandur.org/logfmt) by default. Use these configuration options to change both the logging formats as well as the destination of logs.

import Tabs from '@theme/Tabs';

<Tabs defaultValue="stdoutlogfmt" values={[
  { label: 'Logfmt to Stdout', value: 'stdoutlogfmt', },
  { label: 'JSON to File', value: 'filejson', },
]}>

import TabItem from '@theme/TabItem';

<TabItem value="stdoutlogfmt">

```yaml
logger:
  level: INFO
  format: logfmt
  add_timestamp: false
  static_fields:
    '@service': benthos
```

</TabItem>
<TabItem value="filejson">

```yaml
logger:
  level: WARN
  format: json
  file:
    path: ./logs/benthos.ndjson
    rotate: true
```
